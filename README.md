# Ray Tracer in C

A recursive ray tracer written in C capable of rendering spheres and triangle meshes
with lighting, shadows, reflections, refraction, and texture mapping.

# Features

• Sphere and triangle intersection 

• Blinn-Phong illumination model 

• Hard shadows using shadow rays 

• Reflection and refraction 

• Texture mapping 

• Depth cueing

• Scene parsing from custom input files

# Instructions

1. Compile with make

2. Insert scene details into file.txt

3. Run raytracer.c, and convert file.ppm into a JPG or a PNG
